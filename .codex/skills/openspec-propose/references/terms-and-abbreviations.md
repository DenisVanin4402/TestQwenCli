## Термины

| Термин                            | Определение                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Промт                             | Запрос, команда или набор инструкций, которые пользователь передаёт программе с AI для выполнения определённой задачи                                                                                                                                                                                                                                                                                                                                                                                               |
| System promt                      | Промт, который задается в AI агенте и хранится в дистрибутиве. Может ограничивать User promt                                                                                                                                                                                                                                                                                                                                                                                                                        |
| User promt                        | Промт, который задает пользователь                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| Graceful shutdown                 | "Грамотное завершение" — это процесс остановки сервиса таким образом, чтобы завершить обработку текущих запросов пользователей и сохранить состояние системы перед полным отключением. Это позволяет избежать потери данных, ошибок и проблем с производительностью, обеспечивая плавный переход и минимальное влияние на работу приложения                                                                                                                                                                         |
| SberVault                         | Защищённое хранилище секретов, построенное на платформе HashiCorp Vault                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| SDK (Software Development Kit)    | Комплект разработчика программного обеспечения, представляющий собой набор инструментов, библиотек, документации и примеров кода, предназначенный для облегчения разработки приложений на определенной платформе или технологии                                                                                                                                                                                                                                                                                     |
| JDBC (Java Database Connectivity) | Стандарт Java для подключения и взаимодействия с реляционными базами данных. JDBC предоставляет универсальный интерфейс для работы с любыми СУБД                                                                                                                                                                                                                                                                                                                                                                    |
| In-Memory database (IMDB)         | Высокоэффективное хранилище данных, расположенное в оперативной памяти (RAM), обеспечивающее минимальное время доступа к данным и повышение производительности операций чтения и записи. <br> Оно предназначено для хранения временных данных, подлежащих быстрому изменению и регулярному обращению. <br> Данные организованы в виде коллекций (таблиц) с индексированными полями для эффективного поиска и обработки. Каждая коллекция соответствует типу данных и характеризуется небольшим количеством записей. |
| Креды                             | Учетные данные пользователя (совокупность логина и пароля для какого-либо сервиса)                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| Инжектятся                        | Внедряются                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| Поверхность                       | Интерфейс, обеспечивающий интеграцию и взаимодействие различных сервисов и приложений экосистемы Сбера между собой и с внешними системами. Это механизм, который делает возможным взаимодействие между различными сервисами Сбера и пользователями, обеспечивая удобный и безопасный доступ к ним <br> Подробнее тут: https://confluence.sberbank.ru/pages/viewpage.action?pageId=20736545634                                                                                                                       |
| Rollout Policy                    | Стратегия поэтапного развертывания изменений в продакшн-среде с контролируемым риском, позволяющая постепенно наращивать нагрузку на новую версию и автоматически откатывать её при обнаружении проблем                                                                                                                                                                                                                                                                                                             |
| Сериализация                      | Преобразование текстовых данных в бинарные (двоичные), оптимизированные под быстрый обмен короткими сообщениями                                                                                                                                                                                                                                                                                                                                                                                                     |
