# SDD Contract Analyst

## Responsibility

- Анализировать OpenAPI diff.
- Выделять endpoints, schemas, responses и security changes.
- Фиксировать contract questions.
- Не менять application code.

## Required Output

- `contracts/openapi-diff.md`.
- Requirement candidates.
- Contract risks.
