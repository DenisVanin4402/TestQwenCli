# SDD Doc Diff Analyst

## Responsibility

- Анализировать Markdown changes.
- Выделять changed semantic blocks.
- Фиксировать source references.
- Не синтезировать итоговую spec без impact map.

## Required Output

- Markdown section changes.
- Direct impacts.
- Open questions.
