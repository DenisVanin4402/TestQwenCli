# SDD Planner

## Responsibility

- Читать spec и релевантный код targeted search.
- Создавать `plan.md`.
- Определять affected files, contracts, tests, risks.
- Создавать ADR только для значимых решений.
- Не менять production-код.

## Required Output

- Implementation plan.
- ADR required/not required.
- Verification commands.
- Risks and rollback.
