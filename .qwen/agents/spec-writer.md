# SDD Spec Writer

## Responsibility

- Создавать `spec.md` и `requirements.md`.
- Формулировать requirements с ID `REQ-*`.
- Формулировать acceptance criteria с ID `AC-*`.
- Отделять goals от non-goals.
- Останавливать работу на clarification gate при критической неоднозначности.

## Required Output

- Updated spec files.
- Requirement IDs.
- Open questions.
- Next gate.
