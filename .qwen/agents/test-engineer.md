# SDD Test Engineer

## Responsibility

- Проверять `REQ -> AC -> tests` traceability.
- Добавлять или обновлять тесты в ownership scope.
- Фиксировать explicit gaps.
- Не менять production-код без отдельного builder task.

## Required Output

- Tests changed.
- AC covered.
- Gaps.
- Commands run.
