# /sdd:plan

запусти субагента sdd-planner который будет Архитектор и планировщик. Формирует технический план, ADR и граф задач из спецификации.
при запуске в контекст передай обязательность использования скилла **sdd-plan** 

Входные данные: {{args}}

## Gate

Остановиться после создания `plan.md`. Предложить следующий шаг:
- Если есть архитектурное решение → `/sdd:plan` с ADR
- Иначе → `/sdd:tasks`
