# TestQwenCli

Spring Boot application scaffolded as a Maven project for IntelliJ IDEA.

## Requirements

- Java 21
- Maven 3.9+

## Development

Run tests:

```bash
mvn test
```

Run the application:

```bash
mvn spring-boot:run
```

In IntelliJ IDEA, open the repository root and reload/import `pom.xml` as a Maven project.
