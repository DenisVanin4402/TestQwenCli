Ровный фон

Умеренная смесь sync и async, проверяет базовый throughput, latency и отсутствие накопления очередей.

Sync storm

Высокий sync RPS при небольшом async фоне, нужен для проверки waiters, no slot и резерва под sync.

Async backlog

Большой поток async submit при медленном upstream, показывает рост очереди, retry и возраст задач.

Callback fail

Нагрузка с ошибками callback endpoint, чтобы увидеть неотправленные ответы и retry callback-доставки.

План прогона

UI будет отправлять профиль нагрузки в backend, а backend будет менять runners без рестарта приложения.

PUT /dashboard/api/load/profile

События

Snapshot

Единый снимок состояния для polling-режима и первоначальной загрузки страницы.

GET /dashboard/api/snapshot

Realtime stream

SSE-канал для пульса нагрузки, очередей, слотов и health-событий.

GET /dashboard/api/stream

Load control

Старт, остановка и изменение активного профиля нагрузки.

POST /dashboard/api/load/start POST /dashboard/api/load/stop PUT /dashboard/api/load/profile

Upstream simulation

Latency, jitter, error rate и response size для управляемого поведения внешнего сервиса.

GET /dashboard/api/upstream-simulation PUT /dashboard/api/upstream-simulation

Gateway health

Очереди, callback backlog, waiters, retry, слоты и dispatcher-состояние.

GET /dashboard/api/health

Events

Короткий журнал смены профиля, перегруза, timeout-всплесков и восстановления.

GET /dashboard/api/events
