# Контракты

Эта директория содержит API-контракты и схемы для данной фичи.

## Файлы контрактов

- `openapi.yaml` — REST API спецификация
- `events.asyncapi.yaml` — Async events (если применимо)

## Правила

- Контракты обновляются ДО реализации
- Break changes в контрактах требуют ADR
- Контракты верифицируются через contract-тесты
