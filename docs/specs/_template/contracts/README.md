# Contracts

В этом каталоге хранятся контракты текущей задачи:

- OpenAPI/HTTP schemas;
- AsyncAPI/event schemas;
- JSON examples;
- compatibility notes.
- `openapi-diff.md` для diff-driven анализа API-изменений;
- `dbml-diff.md` для diff-driven анализа изменений модели данных.

Если задача не меняет публичные контракты, зафиксируйте это в `plan.md` и `test-plan.md`.
